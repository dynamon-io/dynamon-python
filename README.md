# Python API for [dynamon.io](https://dynamon.io)


Caches requests.


## Installing

```
pip install dynamon
```


## Using

See `test.py` in this repo.

Visit [nodemon.io/my-random-id](https://nodemon.io/my-random-id)

```py
import dynamon

dynamon('my-random-id', 'line', [['x', 'y'], [0, 0], [1, 1], [2, 0]])
```
